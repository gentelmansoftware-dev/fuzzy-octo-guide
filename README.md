# 🌿 Gentleman Software | Eco-Blockchain Workspace

![License: Apache 2.0](https://img.shields.io/badge/License-Apache 2.0-yellow.svg)
![Status: Live](https://img.shields.io/badge/Status-Live-brightgreen.svg)

**Gentleman Software** es una plataforma de desarrollo de alto rendimiento diseñada bajo la filosofía de "Zero Waste" (cero desperdicio). Este espacio de trabajo integra métricas en tiempo real de Bitcoin y predicciones de ROI mediante IA, todo sobre una infraestructura optimizada para el ahorro energético.

## 🚀 Características Principales

- **Arquitectura Limpia**: Software construido para la máxima velocidad con el mínimo consumo de recursos.
- **Ticker en Vivo**: Seguimiento en tiempo real de BTC/EUR y valor de Satoshis integrado mediante la API de CoinGecko.
- **Predicción de ROI**: Algoritmo de simulación de rendimiento basado en fluctuaciones de mercado.
- **Eco-Sostenibilidad**: Interfaz en negro puro (`#000000`) diseñada para reducir el consumo de energía en pantallas OLED/AMOLED.

## 🛡️ Política de Nodos

Para mantener la eficiencia del ecosistema, los espacios de trabajo siguen una regla de actividad estricta:
> **Inactividad > 365 días:** El nodo entra automáticamente en modo de minado para soporte de la red, transformando recursos pasivos en bloques de valor activo.
